Ejercicio 6

Define la variable z como z = 4.5; luego evalúa:

  1. 0.4z⁴ + 3.1z² − 162.3z − 80.7
  2. \(\frac{z^3 - 23}{\sqrt[3]{z^2 + 17.5}}\)

Ejercicio 7

Define la variable t como t = 3.2; luego evalúa:

  1. \(\frac{1}{2}e^{2t} - 3.8 \ln t^3\)
  2. \(\frac{6t^2 + 6t - 2}{t^2 - 1}\)

Ejercicio 8

Define las variables x y y como x = 6.5 y y = 3.8; luego evalúa:

  1. \(\left(x^2 + y^2\right)^{2/3} + \frac{xy}{y - x}\)
  2. \(\frac{\sqrt{x + y}}{(x - y)^2} + 2x^2 - xy^2\)

Ejercicio 9

Define las variables a, b, c, d como:
c = 4.6, d = 1.7, a = c d², b = (c + a) / (c − d); luego evalúa:

  1. ed-b + 3√(c + a − (ca)d)
  2. \(\frac{d}{c} + \left(\frac{ct}{b}\right)^2 - c^d - \frac{a}{b}\)

Ejercicio 10

Dos identidades trigonométricas son dadas por:

  1. \(\cos^2 x - \sin^2 x = 1 - 2\sin^2 x\)
  2. \(\frac{\tan x}{\sin x - 2 \tan x} = \frac{1}{\cos x - 2}\)

Para cada parte, verifica que la identidad es correcta calculando los valores de ambos lados de la ecuación, usando x = π / 10.
